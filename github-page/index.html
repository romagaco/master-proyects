Hero Web design

¿Qué es un Hero?

Un hero es un banner un poco más grande, desplegado en el encabezado de un sitio web; el primer elemento que verán los visitantes. Para “engancharlos”, lleva fotos, videos, gráficos o ilustraciones, junto a algún enunciado o Llamado a La Acción
